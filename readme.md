<h1>Simple C++ http-rest fileserver</h1>
<i>Build with CMake, stdc, json in CLion </i>

<h2>Feautures:</h4>
1) REST GET file directories<br>
2) REST DELETE files

<h4>Usage:</h4>
1) Compile project<br>
2) Run with terminal
````sh
./rest-fileserver <port> <server_root_dir>
````
